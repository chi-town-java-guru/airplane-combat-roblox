## Publishing Steps

### Step 1: Final Testing
1. Open Roblox Studio
2. Load your game
3. Test all features thoroughly
4. Check for any errors in Output window
5. Test with multiple players if possible

### Step 2: Game Configuration
1. Go to **Game Settings** (gear icon)
2. Set **Basic Info**:
   - Name: "Airplane Combat Game"
   - Description: "Fight enemies inside an airplane! Use punches, kicks, and captures to eliminate threats."
   - Genre: Fighting
3. Set **Permissions**:
   - Max Players: 8
   - Server Type: Standard
4. Set **Monetization**:
   - Enable game passes
   - Configure developer products

### Step 3: Create Assets
1. **Game Icon**: 512x512 PNG image
2. **Thumbnails**: Multiple 1920x1080 images showing gameplay
3. **Game Badge**: Create achievement badges

### Step 4: Publish to Roblox
1. Click **Publish to Roblox** (File > Publish to Roblox)
2. Choose **Create new game**
3. Fill in game details
4. Set visibility to **Public**
5. Click **Create**

### Step 5: Post-Publish Setup
1. Configure game badges
2. Set up leaderboards
3. Configure game analytics
4. Test published version

## Troubleshooting

### Common Publishing Issues
1. **Game won't publish**
   - Check file sizes
   - Verify asset IDs
   - Check internet connection
   - Try publishing in parts

2. **Players can't join**
   - Check server settings
   - Verify game is public
   - Check max player limit
   - Test with different accounts

3. **Monetization not working**
   - Verify game passes are approved
   - Check developer product IDs
   - Test purchase flow
   - Check Robux balance

### Performance Issues
1. **High latency**
   - Optimize scripts
   - Reduce part count
   - Use streaming
   - Optimize network calls

2. **Low FPS**
   - Reduce particle effects
   - Optimize lighting
   - Use LOD (Level of Detail)
   - Profile performance
