# Roblox Airplane Combat Game

A Roblox game where players fight bad people inside an airplane. Players can kick enemies out of the plane, kill them, or capture them until the plane lands.

## Game Mechanics
- Combat system with fighting, kicking, and capturing
- Enemy AI for bad people
- Win/lose conditions
- Player movement inside airplane
- Game state management
