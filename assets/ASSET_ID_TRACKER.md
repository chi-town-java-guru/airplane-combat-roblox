## üîÑ **Upload Instructions**

### **Step 1: Open Roblox Studio**
1. Launch Roblox Studio
2. Create a new place or open existing project
3. Go to **Toolbox** tab

### **Step 2: Upload Audio Files**
1. Click **Create** ‚Üí **Audio**
2. Select each `.mp3` file from `assets/audio/`
3. Give them the **Asset Names** listed above
4. **Copy the Asset ID** from the URL or properties
5. **Paste the ID** in the table above

### **Step 3: Upload Image Files**
1. Click **Create** ‚Üí **Image**
2. Select each `.png` file from `assets/images/`
3. Give them the **Asset Names** listed above
4. **Copy the Asset ID** and update the table

### **Step 4: Update Status**
Change ‚è≥ to ‚úÖ when each asset is uploaded successfully.
